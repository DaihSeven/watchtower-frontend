# Comandos no git:
```bash
//clonar repositório
git clone https://github.com/DaihSeven/watchtower-frontend.git

//abrir pasta
cd watchtower-frontend


//ir para o vscode
code .

//instalar dependências
npm install

//rodar projeto
npm run dev

```

Open [http://localhost:3001](http://localhost:3001)
